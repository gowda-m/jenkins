# Jenkins CI/CD Pipeline – Apache Static Website Deployment (Local)

## Overview
This project demonstrates a CI/CD pipeline setup using Jenkins to automatically deploy a static website to an Apache server on a local machine.

The pipeline pulls code from GitHub and deploys it to Apache’s document root directory, making the website accessible via browser.

---

## 🛠️ Technologies Used
- Jenkins (CI/CD Automation)
- Apache HTTP Server
- Git & GitHub
- Linux (Local System)

---

## Architecture

Local System → GitHub → Jenkins → Apache Server → Browser
